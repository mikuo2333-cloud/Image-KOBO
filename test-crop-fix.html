裁剪窗口修复测试

问题描述

裁剪窗口在拖动到画布边缘时会被"吸住"，当鼠标继续移动时，窗口不会跟随鼠标移动。

修复方案

在边界约束检查后，更新鼠标偏移量，确保当裁剪窗口碰到边界时，鼠标与窗口的相对位置保持一致。

// 修复前
if (targetX < 0) {
    targetX = 0;
}

// 修复后
if (targetX < 0) {
    targetX = 0;
    // Update mouse offset to maintain relative position
    mouseOffset.x = mouseX - targetX;
}

测试方法

  1. 打开主应用程序 (index.html)
  2. 上传一张图片
  3. 选择"裁剪"工具
  4. 创建一个裁剪选区
  5. 尝试将裁剪窗口拖动到画布边缘
  6. 验证裁剪窗口是否能正常跟随鼠标移动，而不是被边缘"吸住"

预期结果

裁剪窗口应该能够平滑地拖动到画布边缘，并且在到达边缘后，即使鼠标继续移动，窗口也应该保持在边缘位置，而不是"跳回"或"吸住"。
